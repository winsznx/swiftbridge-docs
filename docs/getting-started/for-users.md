# User Guide

Welcome to the SwiftBridge User Guide! This guide will walk you through everything you need to know to use SwiftBridge effectively.

## Using the Telegram Bot

The quickest way to get started is with our Telegram Bot.

1. **Start the Bot**: Click [@swiftbridgebot](https://t.me/swiftbridgebot) and click "Start".
2. **Register**: Type `/register <your_username>` to create your account.
3. **Explore**: Use `/help` to see all available commands.

### Key Commands

- `/buy`: Buy crypto with Naira
- `/sell`: Sell crypto for Naira
- `/swap`: Swap between tokens
- `/send`: Send crypto to a Telegram username
- `/balance`: Check your portfolio

## Using the Mini App

For a more visual experience, try our Mini App.

1. **Access**: Go to [app.swiftbridge.app](https://app.swiftbridge.app).
2. **Connect Wallet**: Click "Connect Wallet" and choose your preferred wallet (e.g., MetaMask, Coinbase Wallet).
3. **Dashboard**: View your assets and recent transactions.

### Features

- **Swap Interface**: A familiar DEX interface for swapping tokens.
- **Send Form**: Easily send assets to other users by username.
- **Transaction History**: Detailed view of your past activities.

## Safety First

- Always verify the username before sending funds.
- SwiftBridge will never ask for your private keys.
- Ensure you are interacting with the official bot and app.
